# “公开成长” 
你是否经常浏览朋友圈？你朋友圈的大部分内容都是什么？生活，日常，flag，抱怨，感受，旅游，游戏......或许朋友圈还有更强大的用处：**通过”公开成长“让你成为你想成为的模样**   

无论你之前是怎样的，你都可以通过社交媒体来伪装自己。通过片面的信息让外界误认为你是一个”**“的人。”公开成长“是一种强大的策略：将你的目标、学习过程、进步与挑战公开分享，从而增加责任感、获得外部反馈，并通过社交动态强化你的信念与行动，最终加速“显化”。   
如果你需要健身却迟迟没有开始·，你可以选择先批一张自欺欺人的肌肉照     
如果你想学好c++，你可以分享关于c++的见解，技术，甚至一张c++书本的封面  
如果你想拥有良好的身材，你可以发一张无论批的多假的美美的露腿照

这一切的意义都是误认为，让周围的人误认为你是个”专家“，让自己误认为自己是个”专家“。

# 秘诀：行动    
人类非常在意别人如何看待自己。我们常常通过外界评价来定义自己。 虚荣通常被视为一种罪过。但我们也可以利用它，将其转化为强大的动力。通过塑造一种外部形象，然后努力践行。你可以称之为虚荣、骄傲，或者其他什么。听起来很肤浅，但它确实有效。   
你一直在网上写作公开你的看法，周围的人可能一开始会有各种各样的想法在他们眼中你可能是优秀，高认知，擅长写作的人，实际上并不是。你内心隐隐害怕自己的公开承诺被揭穿，接着你会采取行动来迎合这种看法。其次，你会开始相信自己。你的自我认知会发生变化，以匹配外部认知。你的内驱力开始被无限放大，你会自主开始浏览文章，寻找灵感，开始阅读以寻求更优秀的作品。总有一天，不知不觉中，你会想： 哦，糟了，看来我真的成真了，因为所有这些正经人都认为我正经。
要想成为你想成为的人，首先要让别人这样看待你。开始展现那种人的特质。公开谈论你想成为那样的人。然后，神奇的事情自然而然就发生了。  
## 社会镜像效应(Looking-Glass Self） 
当你公开分享成长内容，周围人开始把你标签为“优秀、高认知、擅长某事的人”，这个“镜子”反射回来，会改变你的自我认知。你会感到“那感觉很好”，从而激励自己不断学习、实践。  
## 承诺一致性原理（Commitment and Consistency）  
一旦公开了某种形象（比如“写作高手”或“AI专家”），大脑会驱使你保持一致。你内心会隐隐害怕“被揭穿”，于是采取行动来迎合这种看法——疲惫时仍去阅读、写作、尝试新工具。    
##  预言自我实现（Self-Fulfilling Prophecy）  
别人对你的期望，变成了你对自己的要求。你公开宣称“我在成为XX”，周围人相信了，你也就“必须”成为那样的人。这形成正向循环：外部期望 → 内在信念 → 持续行动 → 现实强化。

要想成为你想成为的人，首先要让别人这样看待你。开始展现那种人的特质。公开谈论你想成为那样的人。然后，神奇的事情自然而然就发生了。

# 陷阱  
我并不主张频繁浏览社交媒体，对于充斥着同龄人美好生活的朋友圈，至少对我来说：嫉妒使我面目全非。  
发朋友圈的目的只有一个：获得反馈。  
我们很容易就陷入一个发了就不了了之的陷阱：用低成本的“象征性行动”替代真正的行动，让大脑误以为“我已经在做了”，从而止步于舒适区，俗称”假努力“。解决的方法很简单那就是**行动**，如果你一开始就不打算行动那么我的建议是减少社交媒体的浏览吧，包括但不仅限于抖音，微信，QQ，小红书
