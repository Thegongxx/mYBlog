# 最高學以致用法——樺澤紫苑

> 其1 寫日記／最簡單、效果最好的輸出訓練法

这本书是我输出的开始，时隔半年有余，感触颇多。全文为繁体字，为了买到它我花费了不少功夫，大陆没有买的是台版，现在在网上应该是可以简单获取。最让我印象深刻的是他的纸张质量，非常厚，还是彩印，全文字数不多，章节数量多，单个篇幅较少，适合碎片化阅读。从说，打电话，斥责，感谢到睡觉，哭，笑，时间管理都有涵盖，我在拿到目录的时候有点一惊，这些尝试还要拿来侃侃而谈？现在回头看看书中还提到了建立私人“部落格“ blog （原文就是这写的，哈哈），当时印象不深，现在突然看到可能是冥冥之中自有天意。
