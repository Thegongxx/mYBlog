# 最高學以致用法——樺澤紫苑

> 其1 寫日記／最簡單、效果最好的輸出訓練法

这本书是我输出的开始，时隔半年有余，感触颇多。全文为繁体字，为了买到它我花费了不少功夫，大陆没有买的是台版，现在在网上应该是可以简单获取。最让我印象深刻的是他的纸张质量，非常厚，还是彩印，全文字数不多，章节数量多，单个篇幅较少，适合碎片化阅读。从说，打电话，斥责，感谢到睡觉，哭，笑，时间管理都有涵盖，我在拿到目录的时候有点一惊，这些尝试还要拿来侃侃而谈？现在回头看看书中还提到了建立私人“部落格“ blog （原文就是这写的，哈哈），当时印象不深，现在突然看到可能是冥冥之中自有天意。

# 天空不设限1，2——邓予恒

> 抬头看世界，低头做自己

如果让我最推荐一本书，那毫无疑问是这本，原因很简单它几乎重构了我的认知，第一部是自传，对年轻人的一些建议，以及一些方面的思考，第二部则是对未来的预测。第一部的高度太高了，对于当时的我甚至现在的冲击力是巨大的。评价一本书的好坏其实很简单，那就是它能否匹配你的现状，能否以此带来改变。而这本书的出版颠覆了当时颓废的我，如果你看到这里我以我的人格担保，去买《天空不设限》的第一册！不会错，用5秒钟浏览一下电子版的目录也是不错的选择，因为我相信你会控制不住看下去。当然如果你想要，可以随时来问我要电子版。

# 亲密关系——罗兰·米勒

>因为人们乐意被人喜欢，装作冷若冰霜却只对某个人略微有点兴趣，这是一种默默地吸引人的方法。一味故作清高没有什么作用。真正有用的是选择性地故作清高——也就是说，除了你想吸引的人之外，任何人想要得到你都是可望而不可即的（Walster et al.，1973）。那些能付出代价拒绝大多数人却又能高兴地接纳我们的人，才是最有吸引力的未来伴侣。
